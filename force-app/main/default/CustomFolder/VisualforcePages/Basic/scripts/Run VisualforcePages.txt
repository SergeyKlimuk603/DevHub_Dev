С помощью этой строки введенной в консоли разработчика браузера можно открыть страницу Visualforce с именем "HelloWorld" в контектсе Lightning Experience:

$A.get("e.force:navigateToURL").setParams({"url": "/apex/HelloWorld"}).fire();
