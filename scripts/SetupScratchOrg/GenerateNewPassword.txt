Для генерации пароля для новой Scratch Org нужно после ее создания в терминале запустить команду:

sf org generate password

Полезные команды:
sf org open - открывает org
sf org generate password - сгенерирует пароль для орги
sf org display user - отобразит детали скретча (пароль, username и т.д.)
sf project deploy start  --source-dir force-app --target-org my-scratch --concise - деплоит весь проект
